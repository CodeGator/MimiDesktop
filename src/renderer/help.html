Welcome

This window explains the main interface and how to use each part of the app.

Unlock screen

When the app is locked or you have not yet created a vault, you see the unlock screen.

  • Description — A short line explaining that Mimi Desktop is a local secret keeper; data is encrypted and never sent to the cloud.
  • Master password — Enter the password you chose when setting up the vault. Use the eye icon to show or hide the password.
  • Unlock / Create password — If you already have a vault, the button says Unlock; if not, it says Create password and creates a new encrypted vault. The title and hint text change accordingly.
  • Footer — Reminds you to type your password (avoid pasting on shared computers) and that the password cannot be recovered if forgotten.

Vault: tabs

After unlocking, the vault has three tabs at the top:

  • Notes — Secure notes (name + content).
  • Logins — Saved logins (name, URL, username, password, comments).
  • Options — Theme, data location, backup/restore, password settings, and danger zone.

Click a tab to switch. Only one tab is visible at a time.

Notes tab

Manage your secure notes.

  • Toolbar (first row) — New note (plus icon), Import notes, Export selected, Print selected, Delete selected. Export, print, and delete are enabled only when you have selected one or more notes.
  • Filter row — Search box to filter by name or content; Sort dropdown (name A–Z, Z–A, newest/oldest first); count of notes; Select all / Unselect all when you need to export, print, or delete multiple items.
  • List — Each note shows its name and type. Click a row to edit, or use the Edit (pencil) and Delete (trash) buttons on the right. Check the box on the left to include it in “selected” for export/print/delete.
  • Pagination — If there are many notes, use page size, Previous, and Next to move through pages.

Logins tab

Manage saved logins (passwords and related info).

  • Toolbar (first row) — New login, Import logins, Export selected, Print selected, Delete selected. Same idea as Notes: select items first to enable export/print/delete.
  • Filter row — Search, sort, count, and select all / unselect all, same as Notes.
  • List — Each login shows name and username (or type). Click a row to edit. Use Copy (copy password to clipboard), Edit, and Delete. Check the box to select for export/print/delete.
  • Pagination — Same as Notes when the list is long.

Options tab

App settings and data management.

  • Theme — Switch between Light and Dark. The choice is saved.
  • Data location — Shows where the encrypted vault file is stored. Use Browse to choose a different folder (e.g. a USB drive). Changing it affects where new data is saved; existing data is not moved automatically.
  • Export file security — Reminder that exported files are not encrypted by default; you can add a password when exporting. Keep exports in a trusted place and delete after use.
  • Backup / Restore — Backup data creates a zip of your vault and config in a backups folder. Restore from backup lets you pick a backup file and overwrites current data after confirmation. Use only on trusted storage.
  • Password — Lock after sets how many minutes of inactivity before the app locks. Change master password opens a dialog to set a new password (you must enter the current one).
  • Danger area — Delete all data permanently removes the master password and all notes and logins. You must type “delete everything” to confirm. This cannot be undone.

Menus

At the top of the window (or in the system menu bar on macOS):

  • File — Lock locks the vault and returns you to the unlock screen. Unlock brings the window to the front and focuses the password field if the unlock screen is visible. Exit quits the app.
  • Help — Mimi Desktop Help opens this window. About shows app name, version, and copyright.
